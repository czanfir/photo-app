Photo App

What it has:
1. Authentication system (sign-up, edit, log in/logout)
2. Mail confirmation of new account. Mail API used is Sendgrid
3. Users can upload images
4. Uploaded images are stored in an Amazon S3 bucket, with the help of carrierwave and fog gems
5. Basic image functionality (new, edit, index)
6. Mobile friendly (responsive styling)
